# 📡 MDDE102 Sinyal İşleme - Ders Sunumları

Bu depo, **İstanbul Okan Üniversitesi Meslek Yüksekokulu, Elektronik Haberleşme Teknolojisi** programı kapsamında verilen **MDDE102 Sinyal İşleme** dersine ait sunum dosyalarını (PDF), akademik notları ve bu dokümanların **LaTeX** kaynak kodlarını içermektedir.

## 📌 Ders Hakkında

Dersin temel odağı, ağır ve karmaşık matematiksel formalizmden ziyade, haberleşme zinciri (**ADC => Filtre => Spektrum => Modülasyon**) bağlamında sinyal işleme uygulamalarının pratik ve sezgisel olarak kavranmasıdır. Temel hedefimiz, öğrencilerin modern haberleşme sistemlerindeki sinyal akışını hem donanımsal hem de algoritmik düzeyde teknik bir vizyonla anlayabilmelerini sağlamaktır.
